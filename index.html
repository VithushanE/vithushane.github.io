Loan Approval

BACKGROUND

The goal for this project is to predict whether a potential customer is approved for a loan.

This dataset contains detailed customer information, including personal income, home ownership status, employment length, loan intent, loan amount, and more. The variable that we want to predict, "loan status", indicates whether a customer has defaulted on a loan, with a value of 1 representing a default and 0 representing no default.

RESULTS

I built a Random Forest Classifier model that achieved 95% accuracy.

If you want to know more about the technical details, click on the github Repository below. Alternatively, if you want to test out whether you would get approved, select on the second link!
